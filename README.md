# NDU Voice-Native

Custom TTS with voice cloning, design, and 5+ preset voices.

## Voices
- **Jenny**: Bright, friendly
- **Emma**: Warm, natural
- **Ryan**: British, smooth
- **Aria**: Clear, confident
- **Sonia**: Cheerful, engaging

## Endpoints
- `POST /speak` - Generate speech
- `POST /voice/clone` - Clone from audio
- `POST /voice/design` - Design from description
- `GET /voices` - List voices
- `POST /conversation/speak` - Lightweight chat
